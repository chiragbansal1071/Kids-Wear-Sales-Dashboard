# Kids Wear Sales Performance Dashboard

## Project Overview

Developed a real-time Excel dashboard to monitor sales performance for a kids' apparel manufacturing company across multiple e-commerce platforms.

The dashboard helps identify top-performing and underperforming SKUs and Styles while providing platform-specific insights for business decision-making.

---

## Features

- Real-time automated dashboard
- SKU-wise performance tracking
- Style-wise analysis
- KPI calculation using Power Pivot
- Interactive Pivot Tables
- Platform-specific dashboards
- Conditional Formatting
- Automated reporting

---

## Platforms Covered

- Amazon
- Flipkart
- Myntra
- FirstCry
- Other Marketplaces
